# neon-module

Rust bindings for node.js. Based on the tutorial [here](https://neon-bindings.com/docs/intro)

Install:
```bash
$ sudo npm install --global neon-cli
```

Build:
```bash
$ neon build --release

```
Test:
```js
$ npm test

> neon-module@0.1.0 test /mnt/data/home/az/dev/rust101/neon-module
> node test/test.js
```
